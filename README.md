# Themis — Tattoo Artist · Video Editor · Social Media Content Creator

A dark, minimal, single-page portfolio for a tattoo artist who also works as a video editor and social media creator.

Optimized for:

- Bold typography & tattoo-studio aesthetic
- Clean desktop layout + fully responsive mobile view
- Showcasing both tattoo work (images) and edits (videos)
- Simple booking via Instagram
- Easy customization via one JSON file
